# Geldium Credit Delinquency Risk Dashboard

## 🎯 Project Overview
AI-powered credit delinquency prediction 
system built for Geldium Financial Services
as part of Tata iQ Job Simulation Program.

## 📊 Dashboard Features
- 500 customer records analyzed
- 16% overall delinquency rate detected
- 3 risk tiers: High / Medium / Low
- 6 interactive visualizations
- 4 dynamic filters/slicers

## 🔍 Key Insights Found
- Customers with 5+ missed payments 
  have 22.1% delinquency rate
- Age group 46-60 highest risk at 21.7%
- Unemployed customers show 19.4% rate
- Business card holders unexpectedly risky
